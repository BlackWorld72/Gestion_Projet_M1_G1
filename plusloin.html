Définition du Serious Game

Un serious game, ou jeu sérieux en français, est un jeu vidéo ayant pour but de réaliser un objectif sérieux en utilisant un moyen ludique (le jeu vidéo).
Un serious game est défini en différentes catégories :
- Advergame : Ce type de serious game permet la diffusion d’un message promotionnel ou marketing à travers un jeu.
- Edugame : L’objectif de ce serious game est l’apprentissage de connaissances dans le jeu. En entreprise, ce type de jeu est appelé edutainement.
- Exergame : Serious game se basant sur la logique de simulation. Le but étant l'entraînement à réaliser certaines tâches.
- Newsgame : Serious game permettant le passage d’un message d’information.
- Socialgame : Jeu présent sur les réseaux sociaux incluant le plus souvent un élément de monétisation.
- Healthgame : C’est un jeu dédié à la recherche et la santé.
